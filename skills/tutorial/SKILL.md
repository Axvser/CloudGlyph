# Tutorial — Step-by-Step Tutorials & Getting Started Guides

> **Sub-skill.** Load only when the user asks to write a tutorial or getting-started guide.

## When to Load

- "Write a quickstart tutorial"
- "Create a getting started guide"
- "Write a walkthrough for beginners"
- "Write a from-scratch example"

## Writing Method

1. **Identify the target audience** — Beginner / Intermediate / Advanced
2. **Set prerequisites** — Environment requirements, dependency installation
3. **Step-by-step guidance** — Each step has clear input → action → expected output
4. **End-to-end case study** — Use one complete case to connect all steps
5. **Include complete code** — Provide runnable code snippets at each stage

## Format Suggestions

- Use task lists `- [x]` to track completion progress
- Tag code blocks with language and ensure copy buttons work
- Add Mermaid flowcharts for key steps to illustrate execution flow
- Conclude with a link to the full source code and suggested next reads
